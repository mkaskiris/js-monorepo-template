**Θέμα:** Ζητούμενο για ανάπτυξη πλατφόρμας τηλεκπαίδευσης ενηλίκων (WordPress ή
custom λύση)

Θα ήθελα να συζητήσουμε την ανάπτυξη μιας νέας πλατφόρμας τηλεκπαίδευσης ενηλίκων
για το Yiasemi, με στόχο την προετοιμασία υποψηφίων για:

- Εξετάσεις Δημοσίου στην Κύπρο
- Εξετάσεις διορισίμων εκπαιδευτικών (Κοινό Μέρος & Ειδικό Μέρος)

Τα μαθήματα θα γίνονται **διαδικτυακά μέσω Zoom** , ενώ οι μαθητές θα έχουν πρόσβαση σε
υλικό αυτομελέτης (θεωρία, ασκήσεις, τεστ, mock exams) μέσα από την πλατφόρμα.

**1. Στόχος & Κοινό**

- Ενήλικες που προετοιμάζονται για εξετάσεις Δημοσίου και Διορισίμων.
- Θέλουμε η πλατφόρμα να υποστηρίζει τόσο **σύγχρονη διδασκαλία** (live μαθήματα
  μέσω Zoom) όσο και **ασύγχρονη** (υλικό, βίντεο, τεστ).
  **2. Ρόλοι Χρηστών**
- **Διαχειριστής** (εγώ / γραμματεία Yiasemi)
- **Καθηγητές** (να μπορούν να ανεβάζουν υλικό, να βλέπουν βαθμολογίες μαθητών, να
  διαχειρίζονται courses)
- **Εκπαιδευόμενοι** (πρόσβαση μόνο στα μαθήματα στα οποία είναι εγγεγραμμένοι
- **3. Βασικές Λειτουργίες που χρειάζομαι**

1. **Δομή μαθημάτων / courses**

o Κατηγορίες π.χ.:

```

```

```

```
o Δυνατότητα να ανεβαίνουν οι ηχογραφήσεις των μαθημάτων και να

```

μόλις περάσει η περίοδος πρόσβασης.
```

5. **Responsive σχεδιασμός**

**4. Δύο Τεχνικές Επιλογές που εξετάζουμε**

Θα ήθελα τη γνώμη σου και μια αναλυτική πρόταση/προσφορά για τις παρακάτω δύο
επιλογές:

```
Επιλογή 1: Υλοποίηση σε WordPress (με LMS plugin)
```

- Δημιουργία ή αξιοποίηση WordPress εγκατάστασης (σε subdomain ή στο ίδιο
  domain). [http://www.e-yiasemi.com.cy](http://www.e-yiasemi.com.cy)
- Χρήση κάποιου LMS plugin (π.χ. LearnDash, TutorLMS ή άλλο που θεωρείς
  κατάλληλο).
- Ενσωμάτωση Zoom, συστήματος πληρωμών και διαχείρισης μαθημάτων.
- Ρόλοι χρηστών (admin, teacher, student) με αντίστοιχα δικαιώματα.

- Dashboard για μαθητές (προχωρημένα tests, progress κ.λπ.).

Θα ήθελα:

- να μου προτείνεις ποιο LMS plugin θεωρείς καταλληλότερο,
- το εκτιμώμενο κόστος υλοποίησης,
- ενδεικτικό χρόνο παράδοσης.

```
Επιλογή 2: Πλατφόρμα από το μηδέν (custom λύση)
```

- Custom ανάπτυξη πλατφόρμας (με δική σου πρόταση για τεχνολογία/framework).
- Frontend + backend + admin panel.
- Όλες οι λειτουργίες που περιέγραψα (μαθήματα, Zoom, tests, πληρωμές,
  στατιστικά κ.λπ.).
- Δυνατότητα επέκτασης στο μέλλον (π.χ. νέα είδη σεμιναρίων ενηλίκων,
  συνεργασίες με οργανισμούς κ.λπ.).

Θα ήθελα:

- εκτίμηση κόστους ανάπτυξης και συντήρησης,
- εκτίμηση χρόνου υλοποίησης,
- τυχόν πλεονεκτήματα/μειονεκτήματα σε σχέση με τη λύση WordPress.
